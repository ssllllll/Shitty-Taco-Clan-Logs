Notifier
All commands listed here are to be prefixed with yt notifier or yt notify Example: yt notifier list or yt notify list
add <Youtube channel>
Subscribes to Youtube channel in the current Discord channel Note: Custom urls may not always work, if possible please use the url with the channel Id instead.
formatlive <message / reset>
Sets notification message for livestreams in this server. Placeholders: {channel}, {title}, {url}, {url-long} & {type}
format <message / reset>
Sets notification message in this server. Placeholders: {channel}, {title}, {url}, {url-long} & {type}
formatchannel <youtube channel URL> <message / reset>
Sets notification message for the channel provided. Placeholders: {channel}, {title}, {url}, {url-long} & {type}
formatchannellive <youtube channel URL> <message / reset>
Sets livestream notification message for the channel provided. Placeholders: {channel}, {title}, {url}, {url-long} & {type}
list
Shows all the subscribed to Youtube channels in the server.
remove <Youtube channel / all>
Unsubscribe from Youtube channel in the current Discord channel or all Youtube channels in the server. Note: Custom urls may not always work, if possible please use the url with the channel Id instead.
test
Sends a test upload notification in your current Discord channel.
For quick setup please run the following commands in the text channel where u want your notifications to be sent (without the < >).To add a channel run yt notifier add <CHANNEL URL HERE>. If you wish to change the message sent in the notification use yt notifier format <THE MESSAGE YOU WANT TO SAY>, the placeholders can be found in the message above
